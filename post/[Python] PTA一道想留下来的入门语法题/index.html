[Python] PTA一道想留下来的入门语法题

发布于 2021-08-27 | 分类于 Python | 3分钟 | 710字数

Python入门语法题

本题是我觉得基础题里面非常有意思的一道题目

最终改出来的代码非常简洁

涉及到了知识点都很简单，但比较多，有助于理解活用

字典合并 (40分)

输入用字符串表示两个字典，输出合并后的字典。字典的键用一个字母或数字表示。注意：1和‘1’是不同的关键字！

输入格式

在第一行中输入第一个字典字符串；
在第二行中输入第二个字典字符串。

输出格式

在一行中输出合并的字典，输出按字典序。
"1" 的 ASCII 码为 49，大于 1，排序时 1 在前，"1" 在后。其它的字符同理。

输入样例1

在这里给出一组输入。例如：

{1:3, 2:5}
{1:5, 3:7}

输出样例1

在这里给出相应的输出。例如：

{1:8, 2:5, 3:7}

输入样例2

在这里给出一组输入。例如：

{"1":3, 1:4}
{"a":5, "1":6}

输出样例2

在这里给出相应的输出。例如：

{1:4, "1":9, "a":5}

解题思路和细节处理

本题解题思路不难，主要是处理细节
题目本身就是将两个字典相同键的值相加，合并成一个字典

输入的内容正好可以由eval函数经字符串解析成字典结构

然后通过循环，将一个字典的键值对放入另一个字典中
即存在键则相加值，不存在键则添入键值

最后通过排序函数进行字典序排序，得到结果

涉及到的细节知识有

  • eval字符串解析函数
  • 循环遍历字典
  • sorted函数使用，返回值是经过排序的list
    • lambda表达式，单句函数
    • 字典的items方法返回键值对list
    • ord函数返回ascii码值，type判断类型，str代表字符串
  • dict函数由list建立字典
  • print函数中涉及到replace把空格和字符的'号给消除掉

代码给人一种麻雀虽小，五脏俱全的感觉
在合并的时候并不考虑字典序问题
但是排序需要输入list，然后用lambda表达式来定义排序标准
返回的list,又要转换成字典
最后，因为题目要求，python输出的字典字符串
还得消除掉多余的符号和空格

以上解题方案，不考虑性能

解题代码（不带注释）

def main():
    a = eval(input())
    b = eval(input())
    for key, value in b.items():
        if key in a:
            a[key] += value
        else:
            a[key] = value         
    result = dict(sorted(a.items(), key = lambda x: ord(x[0]) if type(x[0]) == str else x[0]))
    print(str(result).replace("'", "\"").replace(" ", ""))
    return    

main()
