关于 福尔摩猫

这是某个很菜的叫做福尔摩猫的博客
它目前并将长期处于无干货阶段，只有一些闲言碎语

个人状况

福尔摩猫，简称懒猫
一个懒惰的肥宅拖拉怪，擅长持久性睡觉
平日里非常地幼稚，什么都不知道
经常搞砸很多事情，非常需要大佬带
但是一般来说没有大佬愿意带我，所以我很无助
本质上是一个，轻度抑郁，表面极度绝望，内心深入充满希望的人

计算机科学与技术

我不是计算机科学与技术专业的。
因为我没能考进去呀。17年高考之后，我是压分的物理系弱鸡。后来呢我就一直想方设法地转到计科，但是又是惨遭失败。学习物理的两年，其实不苦，只是难受，学不进去。不过也正因为如此，再看别的学科好像又都没那么难了。
再后来我终于转到了信息安全专业，算是打了个计算机的擦边球。不过本质上呢，我还是以计算机科学的道路前进着，安全虽然也是时下新兴且重要的，不过对于安全技术，我就当作是知识储备了。我认为，学习计算机，真的是理论和实践的完美结合。在这里不写应该怎么学，只是提一下计算机学科的知识储备。

  • 能力基础
    中文，我想这必须要说，我觉得全篇精华就是，中文实在是最好的东西
    数学，是学科的主要功底。计算机学科本质上就是应用数学，整个计算机都是个数学模型。
    英语，我想这不用多说了，计算机里面全是英语咋办呐
    一小部分物理知识，主要是电路啊什么的最好是会的，一般是硬件学习基础。
  • 理论基础
    硬件：计算机组成与结构，编译原理，计算机网络
    软件：数据结构，算法分析，操作系统原理，数据库系统原理
    其他：软件工程（这个我学的时候很不好，但是在实践中算是慢慢摸索到了一丢丢）
  • 专业方向，仅自己理解
    计算机科学：图形学，图像处理，人工智能，机器学习，算法设计。。。
    软件开发：WEB前端/后端开发，Android移动开发，测试度量，人机交互。。。
    信息安全：系统安全，网络安全，渗透测试，逆向工程，密码分析。。。
    数据库系统：这里涉及到诸多交叉学科，本质上数据库是能在各大领域发挥作用的。
    其他：目前不太了解。

小结

说了这么多，不是说会很全，理解上可能也有些偏差，我只是单纯地去罗列一下。
能力基础和理论基础，我认为一定是跑不掉的。IT行业发展得很快，所以我觉得，这两项没什么好说的，这是以后不断学习的必要保证，也是系统学习的特征。
专业方向的话，像我受到专业的影响，我感兴趣所以会掌握一些密码学的知识，不感兴趣所以会了解一下安全知识。对于想要从业的领域，应当基于良好的基础后，玩几个框架，然后再向下探索框架的本质。
所以，真正在学习的时候，思想和专业基础知识为重，框架什么的可以现学。背一堆API真没啥好吹的。
至于其他方面，随意，我比较喜欢读历史，积累点人文社科的东西还是不错的。
所以，写这个博客，大部分内容将会是和计算机相关的，也有一些其他的闲扯。
希望由此可以让自己不再卑微颓废吧，早日摆脱弱鸡状态。

MarkDown学习和使用

定义

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。   
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。  
Markdown 编写的文档后缀为 .md, .markdown。

刚接触到的时候，虽然认为它很复杂，但是也认为能够实现很多繁杂的排版。
接触了一会儿之后，打消了我花里胡哨的念头（MarkDown不过关），还是专注于内容吧。
其实它不难，但是有坑。尽管如此，困扰我很久的加粗问题终于被我解决啦。以及快捷键真香啊。
在日后的必要写作当中，再慢慢地习惯它吧。
